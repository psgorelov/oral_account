Тренажер устный счет.


Общее описание.


Программа для тренировки навыка устного счёта детей.
Ученику выдается 50 не сложных примеров, которые, можно решить в уме.
При неправильно решенном примере ученик должен решить еще 10 примеров. 
После решения примеров у ученика смотрится время решения всех примеров.
Примеры создаются автоматически по общему признаку (умножение / деление / сложение / вычитание двузначных цифр, умножение / деление / сложение / вычитание трехзначных цифр)


Возможности программы:
Генерация примеров данного типа
выбор нескольких типов примеров, выбор кол-ва примеров
движок, который дает доп кол - во примеров в случае ошибки (возможное дальнейшее развитие)
секундомер решения примеров
сбор статистики
вывод статистики в виде таблицы 


Типы примеров:
(умножение / деление / сложение / вычитание двузначных цифр, умножение / деление / сложение / вычитание трехзначных цифр)




Интерфейс


1.         Выбор типа примера(умножение / деление / сложение / 
вычитание двузначных цифр, умножение / деление / сложение / вычитание трехзначных цифр);
        Просмотр статистики;
2.         Окно решения примеров:
        Пример, сколько примеров осталось, секундомер)
        Ниже: строка с ответом
3.         Окно статистики
        Таблица с значениями: тип примера, сколько решали, время первого   решения, 
        время последнего решения, разница во времени.
